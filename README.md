# To You

To you is a todo app built with vue and tailwind css. You check it out [here](https://cirqlar.github.io/to-you/).

## Features

A todo has:
- [x] A title
- [x] A date
- [x] A category

A category has:
- [x] A name
- [x] A color

You can:
- [x] Create a todo
- [x] Create a category
- [x] Delete a todo
- [x] Edit a todo
- [x] Complete a todo
- [x] Edit a category

## Checking out the code

Pull the code and run `npm start` to start a local web server.
